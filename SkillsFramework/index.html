    • The SkillsFramework schema defines the structure for a professional skills framework, allowing for flexible modeling of different framework structures and approaches such as the UK government's Digital, Data & Technology professional framework, SFIA, or custom organizational competency models.

    Property Name Type Cardinality Description
    name https://schema.org/Text 1 The name of the skills framework
    description https://schema.org/Text 0..1 A description of the framework's purpose and scope
    version https://schema.org/Text 0..1 Version identifier for the framework
    authority https://schema.org/Text 0..1 The organization or body that maintains this framework
    url https://schema.org/URL 0..1 Official URL for the framework documentation
    datePublished https://schema.org/Date 0..1 When this version was published
    dateModified https://schema.org/Date 0..1 When this version was last modified
    levelType https://schema.org/Text 0..1 Describes how levels are structured (hierarchical, matrix, competency-based)
    levels https://schema.org/Text 0..n Level definitions used in this framework
    proficiencyScale https://schema.org/Text 0..n Proficiency measurements used in this framework
    categoryTypes https://schema.org/Text 0..n Types of groupings used (job-family, domain, discipline, capability-area)
    targetSector https://schema.org/Text 0..1 Industry or sector focus
    scope https://schema.org/Text 0..1 Geographic or organizational scope
