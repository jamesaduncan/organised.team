  • Senior End User Computing Engineer

    • A senior end user computing engineer leads complex end-user computing initiatives and establishes best practices for desktop and mobile computing environments. They mentor junior engineers and drive strategic improvements in user computing experiences.

    Typical responsibilities

    • Lead design and implementation of enterprise end-user computing solutions
    • Establish standards and best practices for desktop and mobile computing
    • Mentor and develop junior end-user computing engineers
    • Drive automation and efficiency improvements in user computing services
    • Lead user experience improvements and accessibility initiatives
    • Collaborate with security teams on end-user device security policies
    • Support strategic planning for end-user computing capabilities

    Required Skills

    • Problem Management Practitioner
    • Programming and Build Practitioner
    • User Focus Practitioner
    • Leadership Working
    • Mentoring Working
    • Communicating between the technical and non-technical Practitioner
    • Stakeholder Relationship Management Practitioner
    • Accessibility Working
    • Agile Delivery Working
