  • Data Engineer

    • A data engineer builds and maintains data infrastructure, pipelines, and systems that enable the collection, storage, processing, and analysis of data at scale. They ensure data is reliable, accessible, and secure for analytical and operational use.

    Typical responsibilities

    • Design and build data pipelines and ETL processes
    • Implement data storage solutions and manage data warehouses
    • Ensure data quality, reliability, and performance of data systems
    • Collaborate with data scientists and analysts to support their data needs
    • Monitor and optimise data infrastructure and processing systems
    • Implement data security and privacy measures
    • Automate data workflows and processes

    Required Skills

    • Data Engineering Working
    • Programming and Build Working
    • Data Ethics and Privacy Working
    • Communicating between the technical and non-technical Working
    • Agile Delivery Working
    • Problem Management Awareness
    • Statistical Methods Awareness
