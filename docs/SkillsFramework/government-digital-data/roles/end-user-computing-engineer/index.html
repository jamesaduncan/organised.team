  • End User Computing Engineer

    • An end user computing engineer manages and supports desktop computing environments, mobile devices, and end-user applications. They ensure users have reliable access to technology tools and services needed for their work.

    Typical responsibilities

    • Manage desktop and laptop computing environments and deployments
    • Support mobile device management and security policies
    • Maintain end-user applications and software licensing
    • Provide technical support and troubleshooting for end-user issues
    • Implement and maintain end-user computing security measures
    • Collaborate with service desk teams on user support activities
    • Support user onboarding and technology training initiatives

    Required Skills

    • Problem Management Working
    • Programming and Build Working
    • User Focus Working
    • Communicating between the technical and non-technical Working
    • Stakeholder Relationship Management Working
    • Agile Delivery Awareness
    • Accessibility Awareness
