  • Senior Technical Writer

    • A senior technical writer leads technical documentation for complex programmes and systems. They mentor junior writers, establish documentation standards, and drive technical communication excellence while ensuring all documentation meets user needs and accessibility requirements.

    Typical responsibilities

    • Lead technical writing for complex, multi-system programmes
    • Develop documentation strategies and information architecture frameworks
    • Mentor and develop junior technical writers and communication capability
    • Establish technical writing standards and quality assurance processes
    • Lead cross-functional collaboration on documentation and knowledge management
    • Drive innovation in technical communication tools and approaches
    • Provide expert guidance on complex documentation challenges

    Required Skills

    • Communicating between the technical and non-technical Expert
    • User Focus Expert
    • Leadership Practitioner
    • Mentoring Practitioner
    • Strategy and Planning Practitioner
    • User Research Practitioner
    • Accessibility Practitioner
    • Agile Delivery Practitioner
