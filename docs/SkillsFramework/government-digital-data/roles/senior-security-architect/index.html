  • Senior Security Architect

    • A senior security architect leads complex security transformation initiatives and provides strategic direction on security architecture. They influence organisation-wide security decisions and serve as a senior subject matter expert on security technologies, risk management, and compliance.

    Typical responsibilities

    • Lead enterprise-wide security transformation programmes and strategic initiatives
    • Define organisation security strategy and target security architecture
    • Establish security architecture governance frameworks and standards
    • Lead complex stakeholder engagement across multiple business areas for security solutions
    • Provide expert consultation on strategic security technology decisions
    • Mentor and develop security architecture capability across the organisation
    • Drive innovation and evaluation of emerging security technologies
    • Lead organisation-wide security risk management and compliance initiatives
    • Collaborate with enterprise architects on cross-cutting security concerns

    Required Skills

    • Security Expert
    • Security Architecture Expert
    • Risk Management Expert
    • Governance Practitioner
    • Strategy and Planning Practitioner
    • Communicating between the technical and non-technical Practitioner
    • Stakeholder Relationship Management Practitioner
