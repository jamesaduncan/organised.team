  • Technical Writer

    • A technical writer creates comprehensive, user-focused documentation for complex digital services and technical systems. They understand user information needs, collaborate with technical teams, and produce accessible documentation that helps users successfully complete their tasks.

    Typical responsibilities

    • Create comprehensive technical documentation for complex digital services
    • Develop user-focused API documentation and developer guides
    • Design information architecture for documentation systems
    • Conduct user research to understand documentation needs
    • Collaborate with developers, designers, and product teams
    • Ensure all documentation meets accessibility and inclusion standards
    • Test and iterate documentation based on user feedback

    Required Skills

    • Communicating between the technical and non-technical Practitioner
    • User Focus Practitioner
    • User Research Working
    • Accessibility Working
    • Agile Delivery Working
    • Strategy and Planning Awareness
