  • DevOps Engineer

    • A DevOps engineer bridges the gap between software development and IT operations, implementing practices and tools to automate software delivery, infrastructure management, and monitoring. They focus on improving deployment frequency, reducing failure rates, and accelerating recovery times.

    Typical responsibilities

    • Design and implement CI/CD pipelines for automated software delivery
    • Manage infrastructure as code using tools like Terraform or CloudFormation
    • Configure and maintain containerisation platforms like Docker and Kubernetes
    • Implement monitoring, logging, and alerting systems
    • Automate deployment processes and environment provisioning
    • Collaborate with development teams to optimise application performance
    • Ensure security best practices in deployment and infrastructure

    Required Skills

    • Programming and Build Practitioner
    • Agile Delivery Working
    • Problem Management Working
    • Communicating between the technical and non-technical Working
    • Stakeholder Relationship Management Awareness
    • Consultancy Awareness
