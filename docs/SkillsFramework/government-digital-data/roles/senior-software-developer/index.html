  • Senior Software Developer

    • A senior software developer designs, develops, and maintains complex software systems, providing technical leadership and mentoring to junior developers while ensuring high-quality code and architectural decisions.

    Typical responsibilities

    • Design and implement complex software solutions with consideration for scalability and performance
    • Provide technical leadership and guidance to development teams
    • Conduct code reviews and ensure adherence to coding standards and best practices
    • Mentor junior and mid-level developers in technical skills and career development
    • Collaborate with architects and stakeholders to define technical requirements and solutions
    • Lead technical decision-making for software design patterns and technology choices
    • Ensure software security principles are applied throughout the development lifecycle

    Required Skills

    • Programming and Build Expert
    • Software Design Practitioner
    • Software Development Expert
    • Agile Delivery Practitioner
    • Version Control Expert
    • Testing Practitioner
    • Mentoring Working
    • Leadership Working
    • Communicating between the technical and non-technical Practitioner
