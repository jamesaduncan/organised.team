  • Junior Technical Writer

    • A junior technical writer creates clear, accessible documentation for digital services and technical systems. They work under guidance to produce user guides, API documentation, and technical content while learning about information architecture and user-focused writing approaches.

    Typical responsibilities

    • Create clear technical documentation for digital services and systems
    • Write user guides, help content, and support materials
    • Support API documentation and developer resources
    • Learn and apply accessibility principles in technical writing
    • Collaborate with developers and designers on documentation needs
    • Maintain and update existing documentation and content

    Required Skills

    • Communicating between the technical and non-technical Working
    • User Focus Working
    • Accessibility Awareness
    • Agile Delivery Awareness
    • User Research Awareness
