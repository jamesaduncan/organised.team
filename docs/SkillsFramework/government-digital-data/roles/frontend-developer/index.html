  • Frontend Developer

    • A frontend developer creates engaging and accessible user interfaces for web applications. They work with modern JavaScript frameworks, ensure responsive design, and collaborate closely with designers and backend developers to deliver high-quality user experiences.

    Typical responsibilities

    • Develop interactive web applications using modern JavaScript frameworks
    • Implement responsive designs that work across devices and browsers
    • Ensure web applications meet accessibility standards and guidelines
    • Optimise frontend performance and user experience
    • Collaborate with UX/UI designers to implement design systems
    • Integrate frontend applications with APIs and backend services
    • Write and maintain automated tests for frontend code
    • Participate in code reviews and contribute to coding standards

    Required Skills

    • Programming and Build Practitioner
    • User Focus Practitioner
    • Accessibility Working
    • Agile Delivery Working
    • Communicating between the technical and non-technical Working
    • Problem Management Working
    • User Research Awareness
