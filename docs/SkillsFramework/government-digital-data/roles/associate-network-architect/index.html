  • Associate Network Architect

    • An associate network architect works under guidance to support network architecture activities. They help design network solutions and contribute to network infrastructure planning while developing their expertise in network technologies.

    Typical responsibilities

    • Support the development of network architecture designs and documentation
    • Assist in network capacity planning and performance analysis
    • Help evaluate network technologies and solutions
    • Contribute to network security and compliance activities
    • Support network troubleshooting and incident response

    Required Skills

    • Network Architecture Working
    • Network Technologies Working
    • Security Working
    • Problem Management Working
    • Communicating between the technical and non-technical Awareness
