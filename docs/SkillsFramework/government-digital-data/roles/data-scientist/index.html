  • Data Scientist

    • A data scientist applies data science methods to turn data into actionable insights, supporting decision making within their organisation.

    Typical responsibilities

    • Apply tools and techniques for data analysis and data visualisation
    • Explore existing and new data using a range of statistical tools and techniques, such as machine learning and predictive analytics
    • Find patterns in data and transform them into organisational insight
    • Make recommendations to address complex problems and to inform strategic and operational decision making

    Required Skills

    • Data Visualisation Working
    • Programming and Build Working
    • Statistical Methods Working
    • Data Engineering Working
    • Communicating between the technical and non-technical Working
    • Data Ethics and Privacy Working
    • Problem Management Awareness
