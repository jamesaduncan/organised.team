  • Associate Data Architect

    • An associate data architect works under guidance to support data architecture activities. They help design and implement data solutions while developing their expertise in data architecture principles and practices.

    Typical responsibilities

    • Support the development of data architecture artefacts and documentation
    • Assist in data modelling activities and data flow design
    • Help evaluate data architecture solutions under supervision
    • Contribute to data governance and standards implementation
    • Support stakeholder engagement on data architecture matters

    Required Skills

    • Data Architecture Working
    • Data Modelling Working
    • Data Engineering Working
    • Communicating between the technical and non-technical Working
    • Problem Management Awareness
    • Data Governance Awareness
