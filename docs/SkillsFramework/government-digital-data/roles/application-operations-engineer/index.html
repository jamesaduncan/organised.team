  • Application Operations Engineer

    • An application operations engineer is responsible for maintaining and supporting applications in production environments. They monitor application performance, troubleshoot issues, and ensure applications meet operational requirements and service level agreements.

    Typical responsibilities

    • Monitor application performance and availability in production environments
    • Troubleshoot and resolve application issues and incidents
    • Perform routine maintenance and updates on production applications
    • Collaborate with development teams to improve application operability
    • Implement and maintain monitoring and alerting systems
    • Support application deployments and releases
    • Document operational procedures and maintain runbooks

    Required Skills

    • Problem Management Practitioner
    • Programming and Build Working
    • Agile Delivery Working
    • Communicating between the technical and non-technical Working
    • Stakeholder Relationship Management Working
    • Consultancy Awareness
