  • Lead Security Architect

    • A lead security architect uses their expert knowledge and experience to direct work addressing high level and complex security scenarios. They support and lead other security architects, set strategic direction, and champion security architecture across the organisation.

    Typical responsibilities

    • Lead and direct the application of security architectural principles, methods, and tools across the organisation
    • Establish security architecture strategy and standards across the entire organisation
    • Provide expert consultancy on complex security transformation challenges
    • Lead cross-organisational initiatives and major security architecture programmes
    • Mentor and develop the security architecture capability and community
    • Collaborate with senior leadership to align security architecture with organisational strategy
    • Champion security architecture best practices and maturity across the organisation
    • Represent the organisation in external security architecture communities and forums
    • Drive digital transformation through innovative security solutions
    • Lead organisation-wide security risk governance and compliance assurance
