  • Trainee Data Scientist

    • A trainee data scientist is inquisitive, curious about data and keen to learn and develop. They work under guidance and supervision to support data science activities.

    Typical responsibilities

    • Be aware of how data science techniques can be used, and have a basic knowledge of those key techniques, such as machine learning
    • Support data exploration and analysis activities under supervision
    • Learn and apply basic statistical and analytical techniques
    • Contribute to team data science projects

    Required Skills

    • Data Visualisation Awareness
    • Programming and Build Awareness
    • Statistical Methods Awareness
    • Communicating between the technical and non-technical Awareness
    • Data Ethics and Privacy Awareness
