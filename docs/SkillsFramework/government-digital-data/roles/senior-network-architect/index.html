  • Senior Network Architect

    • A senior network architect leads complex network transformation initiatives and provides strategic direction on network architecture. They influence organisation-wide network decisions and serve as a senior subject matter expert on network technologies and solutions.

    Typical responsibilities

    • Lead enterprise-wide network transformation programmes and strategic initiatives
    • Define organisation network strategy and target network architecture
    • Establish network architecture governance frameworks and standards
    • Lead complex stakeholder engagement across multiple business areas for network solutions
    • Provide expert consultation on strategic network technology decisions
    • Mentor and develop network architecture capability across the organisation
    • Drive innovation and evaluation of emerging network technologies
    • Collaborate with enterprise architects on cross-cutting network concerns

    Required Skills

    • Network Architecture Expert
    • Network Technologies Expert
    • Security Practitioner
    • Strategy and Planning Practitioner
    • Communicating between the technical and non-technical Practitioner
    • Stakeholder Relationship Management Practitioner
    • Leadership Practitioner
    • Mentoring Practitioner
    • Enterprise Architecture Working
