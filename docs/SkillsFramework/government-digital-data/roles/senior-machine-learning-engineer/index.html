  • Senior Machine Learning Engineer

    • A senior machine learning engineer leads complex ML system development and establishes MLOps practices across the organisation. They architect scalable ML infrastructure, mentor junior engineers, and drive innovation in machine learning deployment and operations.

    Typical responsibilities

    • Architect and lead complex machine learning infrastructure projects
    • Establish MLOps practices and automated model lifecycle management
    • Design scalable ML platforms and deployment architectures
    • Mentor junior ML engineers and provide technical guidance
    • Lead cross-functional teams in ML system development and deployment
    • Drive innovation in ML engineering tools and methodologies
    • Ensure ML systems meet performance, security, and ethical standards

    Required Skills

    • Programming and Build Practitioner
    • Data Engineering Practitioner
    • Statistical Methods Practitioner
    • Data Ethics and Privacy Working
    • Communicating between the technical and non-technical Practitioner
    • Mentoring Working
    • Agile Delivery Working
    • Problem Management Working
