  • Associate Security Architect

    • An associate security architect works under guidance to support security architecture activities. They help design security solutions and contribute to security infrastructure planning while developing their expertise in security technologies and risk management.

    Typical responsibilities

    • Support the development of security architecture designs and documentation
    • Assist in security risk assessments and threat modelling activities
    • Help evaluate security technologies and solutions
    • Contribute to security compliance and governance activities
    • Support security incident response and vulnerability management
    • Assist with security design reviews and assessments

    • Security Architecture Working
    • Risk Management Working
    • Problem Management Working
    • Communicating between the technical and non-technical Awareness
    • Governance Awareness
    • Enterprise Architecture Awareness
