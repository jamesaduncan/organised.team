  • Junior Frontend Developer

    • A junior frontend developer is at the beginning of their career, learning to create user interfaces and user experiences for web applications. They work under close supervision to develop their skills in HTML, CSS, JavaScript, and modern frontend frameworks.

    Typical responsibilities

    • Learn frontend technologies including HTML, CSS, and JavaScript
    • Build simple user interfaces following design specifications
    • Implement responsive layouts that work across different devices
    • Follow accessibility guidelines and best practices
    • Participate in code reviews and learn from feedback
    • Fix bugs and make minor enhancements to existing interfaces
    • Collaborate with designers and backend developers

    Required Skills

    • Programming and Build Working
    • User Focus Working
    • Accessibility Awareness
    • Agile Delivery Awareness
    • Communicating between the technical and non-technical Awareness
    • Problem Management Awareness
