  • Junior Accessibility Specialist

    • A junior accessibility specialist supports the team in ensuring digital services are accessible to all users, including those with disabilities. They work under supervision while developing their skills in accessibility testing, standards, and inclusive design principles.

    Typical responsibilities

    • Conduct basic accessibility testing using automated tools and manual techniques
    • Support accessibility audits of digital services and websites
    • Assist in creating accessible content and design recommendations
    • Help maintain accessibility documentation and guidelines
    • Learn and apply WCAG guidelines and accessibility standards
    • Support user testing sessions with disabled users

    Required Skills

    • Accessibility Working
    • User Focus Working
    • Communicating between the technical and non-technical Awareness
    • Agile Delivery Awareness
    • User Research Awareness
