  • Senior Frontend Developer

    • A senior frontend developer leads complex frontend development projects, mentors junior developers, and drives technical decisions for user interface architecture. They have deep expertise in modern frontend technologies and contribute to establishing development standards and practices.

    Typical responsibilities

    • Lead the development of complex frontend applications and components
    • Architect scalable and maintainable frontend solutions
    • Mentor junior and mid-level frontend developers
    • Drive adoption of frontend best practices and modern development tools
    • Collaborate with design systems teams to create reusable components
    • Optimise application performance and user experience metrics
    • Lead code reviews and establish coding standards
    • Evaluate and recommend new frontend technologies and frameworks
    • Ensure compliance with accessibility and security standards

    Required Skills

    • Programming and Build Expert
    • User Focus Expert
    • Accessibility Practitioner
    • Agile Delivery Practitioner
    • Communicating between the technical and non-technical Practitioner
    • Mentoring Working
    • Problem Management Practitioner
    • User Research Working
    • Consultancy Working
