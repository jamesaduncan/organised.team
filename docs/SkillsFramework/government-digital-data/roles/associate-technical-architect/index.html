  • Associate Technical Architect

    • An associate technical architect works under guidance to support technical architecture activities. They help design technical solutions and contribute to technical infrastructure planning while developing their expertise in technical design and software architecture.

    Typical responsibilities

    • Support the development of technical architecture designs and documentation
    • Assist in technical options analysis and technology evaluation
    • Help design technical solutions and software architectures
    • Contribute to technical design reviews and code quality assessments
    • Support technical delivery and development activities
    • Assist with technical testing and performance optimization

    Required Skills

    • Technical Architecture Working
    • Software Development Working
    • Technical Design Working
    • Programming and Build Working
    • Communicating between the technical and non-technical Awareness
    • Problem Management Working
    • Agile Delivery Working
