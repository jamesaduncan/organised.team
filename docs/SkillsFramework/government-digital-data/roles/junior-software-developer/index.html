  • Junior Software Developer

    • A junior software developer is at the beginning of their career, learning to design, build, and test software applications and systems. They work under close supervision as part of multidisciplinary teams to develop their technical skills and understanding of software development practices.

    Typical responsibilities

    • Learn programming languages and development frameworks under guidance
    • Write simple code with supervision and participate in code reviews
    • Fix bugs and make minor enhancements to existing applications
    • Follow established coding standards and best practices
    • Participate in team meetings and contribute to planning activities
    • Learn testing practices and write basic unit tests
    • Document work and seek feedback from senior team members

    Required Skills

    • Programming and Build Working
    • Agile Delivery Awareness
    • Communicating between the technical and non-technical Awareness
    • Problem Management Awareness
    • User Focus Awareness
