  • Associate Test Engineer

    • An associate test engineer is developing their skills in test automation and engineering practices. They work under guidance to build and maintain test automation frameworks, supporting quality assurance efforts through technical testing solutions.

    Typical responsibilities

    • Learn test automation frameworks and tools under supervision
    • Develop basic automated test scripts for functional testing
    • Support manual testing activities and test execution
    • Assist in maintaining test environments and test data
    • Participate in code reviews and follow testing best practices
    • Document test results and contribute to test reporting
    • Work with development teams to understand system requirements

    Required Skills

    • Programming and Build Working
    • Agile Delivery Awareness
    • Problem Management Awareness
    • User Focus Awareness
    • Communicating between the technical and non-technical Awareness
