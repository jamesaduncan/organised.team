  • Security

    • Security involves protecting government systems, data, and services from cyber threats, unauthorized access, and other security risks through appropriate controls and practices.

    Proficiency Levels

    1. Awareness

      You can understand security principles and best practices, and recognise common security threats and vulnerabilities in government environments.

    2. Working

      You can implement security controls, follow security procedures, and contribute to security assessments and incident response activities.

    3. Practitioner

      You can design security solutions, conduct security assessments, manage security incidents, and ensure compliance with government security standards.

    4. Expert

      You can lead enterprise security strategy, establish security frameworks and policies, and drive security culture across government organizations.
