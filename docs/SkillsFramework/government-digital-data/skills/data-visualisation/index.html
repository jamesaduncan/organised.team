  • Data Visualisation

    • Data visualisation involves presenting and exploring data graphically, identifying trends, patterns and outliers in data and using visualisation to tell a story with data.

    Proficiency Levels

    1. Awareness

      You can recognise good data visualisation design and understand how visualisation can be used to tell a story with data.

    2. Working

      You can create basic visualisations and understand when different chart types are appropriate. You can use common visualisation tools.

    3. Practitioner

      You can design effective visualisations that communicate insights clearly. You understand advanced visualisation techniques and can create interactive dashboards.

    4. Expert

      You can lead visualisation strategy and standards. You can create innovative visualisation solutions and teach others best practices in data visualisation.
