  • User Focus

    • User focus involves understanding user needs, behaviours, and contexts to design and deliver services that work well for the people who need to use them.

    Proficiency Levels

    1. Awareness

      You can understand the importance of user-centred design and recognise when user needs should influence decisions.

    2. Working

      You can gather and interpret user feedback, apply user insights to improve services, and advocate for user needs in team discussions.

    3. Practitioner

      You can lead user-centred design processes, synthesise complex user research into actionable insights, and champion user needs across the organisation.

    4. Expert

      You can establish user-centred culture and practices, lead user experience strategy, and influence organisational decisions based on deep user insight.
