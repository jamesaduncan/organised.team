  • Programming and Build

    • Programming and build involves writing code to build applications, systems and products. This includes designing, coding, debugging, testing and documenting software solutions.

    Proficiency Levels

    1. Awareness

      You can understand basic programming concepts and can read simple code. You understand how software development fits into the wider development process.

    2. Working

      You can design, code and test simple software components under guidance. You can use version control and follow coding standards.

    3. Practitioner

      You can design, code and test code transformation and data processing components. You can work independently on complex programming tasks and mentor others.

    4. Expert

      You can design and build complex systems architecture. You can set technical direction and standards for programming practices across teams.
