    • An organization is a collection of entities that work together to achieve a common goal or goals.

    Beyond the properties specified here, an organisation can also have the properties specified in https://schema.org/Organization.

    Property Name Cardinality Type Description
    supplies 0..n https://schema.org/UserNeed a user need that this organisation satisfies
    demands 0..n https://schema.org/UserNeed a user need this organisation depends upon
    employee 0..n https://schema.org/Person Someone working for this organization. Overrides schema.org/Organization by providing cardinality
