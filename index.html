Organised Team

This is a placeholder for now but Organised Team provides a small number of microdata schemas that are used to describe various elements of the Organised Team platform. The schemas are designed to be used in exchanging data with organised team, and other software platforms.

The following schemas are documented:

  • Enumerated - Defines a datatype that allows for a fixed set of values.
  • Cardinal - An enumerated type with specific cardinality values.
  • Schema - Defines a schema for Schemas
  • Property - a schema for schema properties
  • Organization - a definition for organisations that have and fulfil needs
  • Team - Represents a team as an organization.
  • Policy - a schema for policy definitions
  • Standard - a schema for a standard definition
  • User Need - a scheme representing a need
  • Software - Represents a piece of software as a product
  • SkillsFramework - defines the structure for a professional skills framework
  • Role - defines a specific job role or position within a skills framework
  • Skill - an individual competency or ability within a skills framework
